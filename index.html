Work & Projects

Featured Projects

A selection of self-built tools and applications developed for learning, community use, and creative exploration.

PHP · JavaScript

Suno v25 — Media Player

A self-hosted PHP media player featuring Web Audio API spectrum visualization, drag-and-drop upload, per-track reactions, and a DNA-themed animation engine. Final project for CS50x.

Tone.js · HTML

Raag Player

An interactive browser-based instrument player tuned to Gurugranth Sahib Ji ragas, combining Indian classical music theory with modern Web Audio synthesis.

Python · systemd

PyAudio Daemon

A userspace audio daemon for Debian 13 using asyncio IPC, ALSA backend, a software mixer with soft-clipping, and a systemd service unit for persistent audio routing.

Bash · Motion

Dual-Camera Surveillance Script

A colour-coded Bash automation script managing dual-camera recording via Motion and systemd, with careful MP4-only file handling and service lifecycle management.

Tone.js · HTML

1615-Instrument Synthesizer

A fully browser-based general-MIDI synthesizer covering 1,615 instrument patches, built as a single self-contained HTML file with Tone.js.

PHP · Apache

Local Media Server

A self-hosted Apache + PHP 8.4 + MariaDB stack on Debian 13, powering a suite of creative tools, kirtan players, and audio visualization apps served locally.

About Us

Who We Are

Alfa Centauri Software is an independent software development practice based in Mississauga, Ontario, Canada. Founded by Bhupinder Deol, the practice focuses on practical, handcrafted web applications built without heavy frameworks or build pipelines — just clean code that runs.

Our work spans interactive music tools rooted in Indian classical traditions, security-oriented Linux utilities, CS50x coursework, and bespoke web applications for real-world use.
