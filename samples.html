About Us

Who We Are

Alfa Centauri Software is an independent software development practice based in Mississauga, Ontario, Canada. Founded by Bhupinder Deol, the practice focuses on practical, handcrafted web applications built without heavy frameworks or build pipelines — just clean code that runs.

Our work spans interactive music tools rooted in Indian classical traditions, security-oriented Linux utilities, CS50x coursework, and bespoke web applications for real-world use.
